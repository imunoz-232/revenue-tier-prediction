# Revenue Tier Prediction App

This project predicts whether a transaction belongs to Low, Medium, or High revenue tier using supervised machine learning.

## Features
- Machine Learning model (Logistic Regression)
- Flask API
- Web interface (HTML + JavaScript)
- Real-time predictions

## How to run

1. Clone the repo:
git clone https://github.com/imunoz-232/revenue-tier-prediction.git

2. Install requirements:
pip install -r requirements.txt

3. Run the app:
python app.py

4. Open in browser:
http://127.0.0.1:5000

## Example Prediction
- Input: transaction data
- Output: Revenue Tier (Low / Medium / High)
